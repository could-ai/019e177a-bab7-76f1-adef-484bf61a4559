# Research Presentation Builder

แอปพลิเคชันสำหรับสร้างพรีเซนเทชั่นนำเสนองานวิจัยและวิทยานิพนธ์ รองรับการทำงานทุกแพลตฟอร์ม

## ฟีเจอร์หลัก
- สร้างและจัดการงานนำเสนอ
- จัดการสไลด์ (เพิ่ม/แก้ไข/ลบ)
- รองรับเลย์เอาต์ทั้งบนมือถือและเดสก์ท็อป

## Tech Stack
- Flutter (Dart)
- Material Design 3

## วิธีการใช้งาน
1. ติดตั้ง Flutter
2. รันคำสั่ง `flutter pub get`
3. รันแอปด้วยคำสั่ง `flutter run`

## โครงสร้างโปรเจกต์
- `lib/models/`: โมเดลข้อมูลของพรีเซนเทชั่นและสไลด์
- `lib/screens/`: หน้าจอ UI ต่างๆ (หน้าหลัก, หน้าแก้ไขสไลด์)

---

## เกี่ยวกับ CouldAI
แอปพลิเคชันนี้ถูกสร้างขึ้นด้วย [CouldAI](https://could.ai) เครื่องมือสร้างแอปพลิเคชันด้วย AI ที่สามารถเปลี่ยนข้อความคำสั่งให้เป็นแอปพลิเคชันที่ใช้งานได้จริงบน iOS, Android, Web และ Desktop โดยอัตโนมัติ
